# 🌐 https://www.primesepto.shop

## 🚀 End-to-End Application Development Stack

### 🧩 Backend Technologies
- Java 17
- Spring Boot (3.x)
- Spring MVC
- Spring Data JPA (Hibernate)
- Spring Security

### 🌐 Frontend Technologies
- HTML5
- CSS3
- JavaScript

### 💳 Payment Integration
- Razorpay Payment Gateway
- REST APIs for secure payment processing

### 📧 Email & OTP Services
- Spring Boot Starter Mail (Java Mail Sender)
- OTP-based Authentication System

### 🗄️ Database & Caching
- MySQL (AWS RDS / Local)
- Hibernate ORM
- Redis (Caching & Performance Optimization)

### ☁️ Cloud & Deployment
- AWS EC2 (Application Hosting)
- AWS RDS (Database Management)
- AWS Secrets Manager (Secure Credential Storage)

### 🔐 Security & Configuration
- Environment Variables
- Externalized Configuration (application.properties / YAML)
